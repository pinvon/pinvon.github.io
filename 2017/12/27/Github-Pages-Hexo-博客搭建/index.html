一台主机，两个Github帐号

  • 使用ssh连接到Github，需要有唯一的公钥。如果一台主机上想要使用两个Github帐号，则必须有两个公钥才行，并且区分每个帐号各自使用的公钥。

  • 办法：生成的新密钥，命名不能与已有密钥重复，还要取消Git的全局 用户名/邮箱 设置，为每个仓库单独设置，这样才可以使用不同的帐号，否则都会默认使用全局的帐号。

验证主题

主题配置文件

Scheme

Scheme可以让我们使用不同的外观，并且配置基本可以共用。在_config.yml中寻找scheme关键字，可以看到我们可用的scheme。

Menu

搜索Menu关键字，根据需要显示。

在_config.yml中，每个菜单项所对应的文字显示，根据语言的设定，在languages/下有每种语言的翻译。

但是，此时页面所显示的文字，却未必是我们在文件中所的。如果没有正确显示，则修改站点配置文件_config.yml，把其中的languages显式指定，如指定为zh-Hans。

首页不显示全文

关键字：auto_excerpt；将enable对应项设为true。
